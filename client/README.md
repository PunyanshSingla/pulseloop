# PulseLoop Client 🎨

The frontend of PulseLoop is built with a focus on speed, real-time interactivity, and a premium aesthetic.

## 🚀 Key Features

### 📈 Admin Dashboard
- **Live Analytics**: Real-time KPI tracking for response volume and engagement.
- **Interactive Charts**: Responsive data visualizations for trends and user demographics.
- **Device Breakdown**: Visual reporting of platform distribution (Mobile, Desktop, Tablet).
- **Activity Stream**: Live websocket-powered feed of global polling events.

### 🗳️ Voting & Interaction
- **Step-by-Step Voting**: Focused interface for multi-question polls.
- **Progress Tracking**: Real-time feedback on completion status.
- **Confetti Celebration**: Immediate visual feedback upon successful participation.
- **Public Results**: High-fidelity results view for published polls.

### 🤖 AI Builder
- **Prompt-to-Poll**: Smart interface for generating full polls using Google Gemini.
- **Instant Preview**: Real-time preview of AI-generated content.

### 🔐 User Experience
- **Profile Center**: Avatar customization and linked account management.
- **Auth Flows**: Integrated Better-Auth UI for sign-up, login, and verification.
- **Theme Engine**: Support for premium Dark and Light modes.

## 🛠️ Tech Stack
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS v4, shadcn/ui
- **State Management**: TanStack Query (React Query)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts
